01

مَن يستفيد؟ وكيف؟

كل من يحتاج رقمًا موثوقًا عن اللجوء والنزوح ولا يملك وقتًا (أو رغبة) لمصارعة واجهات REST:

الباحث الأكاديمي

دراسات الهجرة القسرية · الاقتصاد السياسي · دراسات النزاع

يسأل: «هات سلسلة النازحين من السودان 2010–2025 كـ CSV مع تعليم السنوات الشاذة»

يحصل على: بيانات نظيفة جاهزة لـ R أو Stata أو Python، بأكواد ISO موحدة، مع توثيق المصدر للاستشهاد — في ثوانٍ بدل يوم تنظيف بيانات.

صحفي البيانات

تقارير استقصائية · متابعة الأزمات · تدقيق التصريحات

يسأل: «هل فعلًا أوغندا من أكبر خمس دول مستقبلة للاجئين هذا العام؟»

يحصل على: ترتيبًا محدثًا بأرقام رسمية (نعم — ثالث أكبر مستضيف في 2025) + رسمًا جاهزًا للنشر + جملة الإسناد — قبل موعد التسليم.

منظمة الإغاثة

كتابة المنح · مذكرات المانحين · تخطيط البرامج

تسأل: «جهّز مذكرة مانحين عن الوضع في تشاد مع التركيبة العمرية للوافدين»

تحصل على: تقرير Markdown كاملًا — أرقامًا رئيسية، اتجاه عشر سنوات، نسب النساء والأطفال — جاهزًا للتحرير بدل أسبوع جمع يدوي.

الطالب والمعلّم

أبحاث التخرج · العروض التقديمية · فصول علوم البيانات

يسأل: «اشرح لي اتجاه لجوء السوريين آخر عشر سنين برسم بياني بسيط»

يحصل على: شرحًا بلغة بسيطة + رسم Mermaid يلصقه في العرض مباشرة — ومصدرًا يستطيع الرجوع إليه والتحقق منه.

02

لماذا لا يكفي أن «يعرف» الذكاء الاصطناعي الواجهة الرسمية؟

لأنها ملغومة: مفوضية اللاجئين تستخدم أكواد دول خاصة تخالف معيار ISO في ٩٩ دولة من ٢٣٢. كود مصر عندهم ARE — وهو كود الإمارات في بقية العالم. والخطأ لا يُنتج تحذيرًا؛ بل مصفوفة فارغة، فيجيب مساعدك بثقة: «لا توجد بيانات».

هذا المشروع حفظ الألغام كلها مرة واحدة — وكل مساعد يتصل به يرث الحماية مجانًا.

RAW REST

GET /population/?coa=EGY
{ "items": [] }          // silent — wrong code

HUMANITARIAN-MCP

refugee_population({ country: "Egypt" })
{ country: "Egypt", country_code: "EGY",
  year: 2025, refugees: 220,033, … }

03

ثلاث ورشات جاهزة للباحثين

مسارات كاملة من السؤال البحثي إلى بيانات قابلة لإعادة الإنتاج والاستشهاد — بأرقام حقيقية، لا أمثلة وهمية.

١ · دراسة أثر حدث

السؤال: ماذا فعلت حرب أبريل 2023 بأعداد النازحين من السودان؟

أداة تحليل الاتجاه تعطيك السلسلة كاملة، الانحدار الخطي، وتعلّم سنوات الكسر إحصائيًا — ثم تصدّر CSV نظيفًا يدخل مباشرة في نموذج الانقطاع الزمني عندك. 

# in the conversation
trend_analysis({ country: "Sudan", role: "origin", year_from: 2010 })
→ anomalies: [{ year: 2023, changePct: +78.8, zScore: 2.6 }]

export_data({ dataset: "population", country: "Sudan",
              role: "origin", format: "csv", year_from: 2010 })

# then in Python
import pandas as pd
df = pd.read_csv("sudan.csv")     # country_code, year, refugees…
df["post_war"] = (df.year >= 2023).astype(int)
# interrupted time-series as usual

٢ · مقارنة سياسات اللجوء

السؤال: كيف تختلف نسب قبول طلبات اللجوء بين الدول المستقبِلة؟

أداة القرارات تحسب نسبة الاعتراف تلقائيًا (معترف + حماية تكميلية ÷ القرارات الموضوعية) لكل سنة ولكل دولة — المتغير التابع في أدبيات سياسات اللجوء، بلا حسابات يدوية. 

asylum_decisions({ country: "Egypt", year_from: 2019 })
→ 2023: recognized 9,651 · rejected 1,977 · closed 4,375
  recognition_rate: 83.0%

# repeat per country, or batch-compare:
compare_countries({ countries: ["Egypt","Jordan","Uganda"],
                    metric: "asylum_seekers" })

٣ · التركيبة السكانية للاستجابة

السؤال: ما حصة النساء والأطفال بين اللاجئين في مصر؟ — الرقم الذي تُبنى عليه برامج الحماية والتعليم وتُكتب به المنح.

ناتج حقيقي من بيانات 2025، مع رسم جاهز للتقرير بأي صيغة تطلبها. 

demographics({ country: "Egypt" })
→ total: 1,098,306 · female: 54.2% · children (0–17): 39.1%

generate_chart({ countries: ["Egypt"], format: "svg" })
# → SVG ready to drop into the report

الدليل الكامل بأمثلة R و Python وقواعد الاستشهاد: docs/for-researchers.md ↗ · وكل رقم يخرج ومعه سنة المرجع والمصدر — قابل للتحقق دائمًا من Refugee Data Finder ↗

04

شاهِده يُجيب.

أنت تسأل بالعربي، المساعد يختار الأداة، والخادم يتكفل بالباقي — والأرقام أدناه حية من UNHCR.

claude — humanitarian-mcp متصل

أنت ‹ قارن أعداد اللاجئين في مصر والأردن خلال آخر خمس سنوات.

⚙ compare_countries({ countries: ["Egypt","Jordan"] }) · 1.2s

| Year | Egypt   | Jordan  |
| 2021 | 280,686 | 712,823 |
| 2023 | 240,507 | 684,066 |
| 2025 | 220,033 | 436,226 |

claude › تستضيف الأردن قرابة ضعف عدد اللاجئين المسجلين في مصر، لكن الرقمين تراجعا حتى 2025 — الأردن بحدة (−32% منذ 2021)… المصدر: UNHCR.

05

جاهز في دقيقتين.

أضِف هذا إلى claude_desktop_config.json وأعد تشغيل Claude — ثم اسأل بلغتك.

{
  "mcpServers": {
    "humanitarian": {
      "command": "node",
      "args": ["/path/to/humanitarian-mcp/dist/index.js"]
    }
  }
}

أمر واحد يسجّل الخادم لكل الجلسات:

claude mcp add humanitarian -- node /path/to/humanitarian-mcp/dist/index.js

ثم جرّب:

  • « اعمل تقريرًا إنسانيًا كاملًا عن السودان »
  • « صدّر قرارات اللجوء في الأردن 2018–2025 كملف CSV »
  • « أي الدول شهدت قفزات نزوح شاذة منذ 2015؟ »

نقطة HTTP عديمة الحالة — كل طلب مستقل، وتعمل خلف أي load balancer.

npm run dashboard   # POST /mcp على المنفذ 8642

curl -s -X POST http://localhost:8642/mcp \
  -H 'content-type: application/json' \
  -H 'accept: application/json, text/event-stream' \
  -d '{"jsonrpc":"2.0","id":1,"method":"tools/call","params":{
        "name":"top_host_countries","arguments":{"limit":5}}}'

بدون عميل MCP؟ لوحة العرض المدمجة تعرض المزوّدين والكتالوج وسجلات حية — وتشغّل أي أداة من المتصفح.

npm run dashboard
# → http://localhost:8642

إعدادات اختيارية: HMCP_CACHE=sqlite يحفظ البيانات بين الجلسات · HMCP_OFFLINE=1 يعمل من الكاش فقط — مفيد في الميدان حيث لا اتصال.

06

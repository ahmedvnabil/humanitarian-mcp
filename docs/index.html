خادم مفتوح المصدر لبروتوكول Model Context Protocol

بياناتُ اللجوء والنزوح،
بين يدي مساعدك الذكي.

اسأل Claude أو Cursor أو أي عميل MCP: «قارن أعداد اللاجئين في مصر والأردن» — فيصلك جدول موثّق من بيانات مفوضية اللاجئين (UNHCR) الممتدة عبر ٧٥ عامًا. بلا تعقيدات REST، بلا أكواد دول ملغومة، بلا مفاتيح API.

git clone https://github.com/ahmedvnabil/humanitarian-mcp && cd humanitarian-mcp && npm i && npm run build
17 أداة دلالية 7 قوالب جاهزة 99 اختبارًا ناجحًا 0 مفاتيح مطلوبة يعمل دون اتصال بعد التخزين المؤقت

الواجهة الرسمية ملغومة.
نحن نزعنا الألغام مرةً واحدة.

تستخدم واجهة UNHCR أكوادَ دولٍ خاصة بها تخالف معيار ISO في ٩٩ دولة من أصل ٢٣٢.

كود مصر عندهم ARE…
وهو نفسه كود الإمارات في ISO!

والأدهى: الخطأ لا يُنتج رسالة خطأ، بل مصفوفة فارغة — فيُبلغ مساعدُك بثقة أن «لا بيانات»، وهو مخطئ. هذا المشروع يحفظ كل هذه الفخاخ مرة واحدة، خلف أدوات بأسماء بشرية.

RAW REST

GET /population/v1/population/?coa=EGY

{ "items": [] }   // silent — wrong code

HUMANITARIAN-MCP

refugee_population({ country: "Egypt" })

{ country: "Egypt", country_code: "EGY",
  year: 2025, refugees: 220,033, … }
موحَّد. حقول ثابتة و ISO3 عبر كل مزوّد حالي وقادم.
قراءة فقط. كل أداة معلَّمة read-only؛ لا شيء يكتب للمصدر.
مهذَّب. حدود معدل، إعادة محاولة، تخزين مؤقت بـ ETag.
موثَّق. إسناد UNHCR يرافق كل رقم يخرج.

شاهِده يُجيب.

حوار حقيقي: أنت تسأل بلغتك، المساعد يختار الأداة، والخادم يتكفل بالسباكة. الأرقام أدناه بيانات UNHCR حية — ليست للزينة.

claude — humanitarian-mcp متصل

أنت ‹ قارن أعداد اللاجئين في مصر والأردن خلال آخر خمس سنوات.

⚙ tools/call compare_countries { countries: ["Egypt","Jordan"] } · 1.2s

| Year | Egypt   | Jordan  |
| ---- | ------- | ------- |
| 2021 | 280,686 | 712,823 |
| 2022 | 294,632 | 697,761 |
| 2023 | 240,507 | 684,066 |
| 2024 | 238,014 | 643,641 |
| 2025 | 220,033 | 436,226 |

claude › تستضيف الأردن قرابة ضعف عدد اللاجئين المسجلين في مصر، لكن الرقمين تراجعا حتى 2025 — الأردن بحدة (−32% منذ 2021)… المصدر: UNHCR Refugee Data Finder.

جاهز في دقيقتين.

أضِف هذا إلى claude_desktop_config.json، أعد تشغيل Claude، ثم اسأل بلغتك مباشرة.

{
  "mcpServers": {
    "humanitarian": {
      "command": "node",
      "args": ["/path/to/humanitarian-mcp/dist/index.js"]
    }
  }
}

أمر واحد يسجّل الخادم لكل الجلسات:

claude mcp add humanitarian -- node /path/to/humanitarian-mcp/dist/index.js

ثم داخل Claude Code جرّب:

  • « اعمل تقريرًا إنسانيًا كاملًا عن السودان »
  • « ما أكثر ١٠ دول استضافةً للاجئين هذا العام؟ »
  • « صدّر قرارات اللجوء في الأردن 2018–2025 كملف CSV »

نقطة HTTP عديمة الحالة — كل طلب مستقل بذاته، وتتوزّع خلف أي load balancer.

npm run dashboard   # POST /mcp على المنفذ 8642

curl -s -X POST http://localhost:8642/mcp \
  -H 'content-type: application/json' \
  -H 'accept: application/json, text/event-stream' \
  -d '{"jsonrpc":"2.0","id":1,"method":"tools/call","params":{
        "name":"top_host_countries","arguments":{"limit":5}}}'

بدون عميل MCP؟ لوحة العرض المدمجة تعرض المزوّدين وصحتهم والكتالوج كاملًا وسجلات حية — وتشغّل أي أداة من المتصفح مباشرة.

npm run dashboard
# → http://localhost:8642

إعدادات اختيارية: HMCP_CACHE=sqlite تحفظ البيانات بين الجلسات · HMCP_OFFLINE=1 يعمل من الكاش فقط · القائمة الكاملة في .env.example ↗

١٧ أداة — كلها قراءة فقط

  • search_country— يفهم «سوريا» ويحوّلها SYR
  • country_profile— لمحة شاملة بنداء واحد
  • compare_countries— حتى ٥ دول جنبًا لجنب
  • refugee_population— سنويًا، بترقيم صفحات
  • demographics— العمر × الجنس
  • latest_statistics— أحدث الأرقام
  • asylum_applications— طلبات اللجوء سنويًا
  • asylum_decisions— مع نسبة القبول
  • trend_analysis— ميل، R²، سنوات شاذة
  • forecast— إسقاط بتحذير صريح
  • top_host_countries— ترتيب الدول
  • generate_chart— Chart.js/Vega/Mermaid/SVG
  • generate_map— خريطة GeoJSON
  • generate_country_report— تقرير Markdown كامل
  • export_data— CSV/JSON/MD/GeoJSON
  • get_metadata · provider_health
